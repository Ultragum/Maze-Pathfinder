# A Maze Pathfinder with a wxWidgets-powered GUI

This project is a maze pathfinder that implements the A* pathfinding algorithm in C++. It features a GUI that was created with the wxWidgets library.
The maze itself it dynamic, allowing the user to edit the size, obstacles, and the two points to pathfind between.

# Building

Download and build [wxWidgets](https://wxwidgets.org/downloads/), update the additional include and additional library directories in Visual Studio to point towards the built library,
and then finally build this program.

# Running

To run the program, either build and run it via Visual Studio, or download the release version and run the executable.
